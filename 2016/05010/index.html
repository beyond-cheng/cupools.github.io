Git 配置

  • 碎屑

每次跟别人说 Git 要怎么使用的时候都要手把手教怎么配置 (为什么你不自己谷歌咧)，拖到现在终于落实到文档了，以后就丢这个过去就行了虽然以后也没什么机会了。

安装

官方下载地址，下载安装即可

配置

配置 SSH key

这里通过 ssh 管理代码仓库，需要配置 SSH key; 如果通过 http 管理代码仓库，则直接使用 Github 的账号密码即可。

1. 检查 SSH key 是否存在

2. 备份已有的 key, 如果有的话~

3. 生成 SSH key

之后会提示输入路径，默认 ~/.ssh, 回车确认；之后会提示创建密码，之后使用 SSH key 的过程中会用到，输入后回车确认，也可以直接回车不使用密码

(发现 OS X 需要 Xcode 支持)

创建完成之后， 在 ~/.ssh 里面找到 id_rsa.pub 这个文件，即是 SSH key.

4. 将 SSH key 添加到 GitHub

登录到 GitHub ，Settings->SSH and GPG keys->New SSH key, 将生成的 key(id_rsa.pub 文件)内容复制到输入框中，填写标题后保存即可。

5. 验证

出现上述提示信息说明连接成功 (Github 并不提供 shell 登陆但已经连接成功)

设置个人信息

主要是个人姓名和邮箱，在提交代码的时候会带上这部分信息。
