css内置字体族，所有css兼容的浏览器有效。几种字体的区别

font feature
serif 有明显衬线(默认Times New Roman)
sans-serif 无衬线(默认Arial)
monospace 每个字母有相同的宽度，显示代码有用(默认courier)
cursive 模拟手写风格，常用于标题(常用Comic Sans)
